Simon Says

When the game begins, the computer will start a sequence of buttons. When it is complete it will be up to you to repeat that sequence by clicking the buttons in the correct order. If you are successful, the computer will repeat the same sequence with one additional step and it will be up to you to repeat it again. See how long you can go, click New Game to begin.
